<stereo-img>: Button Rendering Tests

Default Behavior (no controlslist)

Expect to see the VR button and all three static mode buttons (static, wiggle, and anaglyph).

controlslist="vr"

Expect to see only the VR button.

controlslist="static wiggle"

Expect to not see the VR button, and see only the static and wiggle buttons in the static mode group.

controlslist="wiggle static" (order insensitive)

Order insensitive, expect to not see VR, and see only the static and wiggle buttons in the static mode group.

controlslist="vr static"

Expect to see the VR button and no static mode group, since there is only one choice available.

controlslist="vr static wiggle anaglyph"

Expect to see the VR button and the static, wiggle, and anaglyph buttons in the static mode group.
